debuild.me

debuild.me is a buildd network, run by friends, to help automagically lint, build and test Debian packages.

The name comes from the Debian tool of the same name -- debuild, which builds Debian packages. Although we don't use `debuild' under the hood, it's still a mighty fiting name.
